# 🎓 IHK Anwendungsentwicklung - Prüfungsvorbereitung

## ✅ FOKUSSIERTE LERNFELDER (9 Fächer)

### BLOCK 1: KERNKOMPETENZEN (60% der Prüfung)

#### ⭐⭐⭐ LF2: Programmiergrundlagen (150h)
**SUPER WICHTIG - Basis für alles!**
- Python, Java, C# Syntax
- OOP: Klassen, Vererbung, Polymorphismus
- Exception Handling
- Design Patterns (Singleton, Factory, Observer)

**Prüfungsfokus:** OOP-Prinzipien, Fehlerbehandlung

---

#### ⭐⭐⭐ LF3: Datenbanken (100h)
**SUPER WICHTIG - Wird definitiv abgefragt!**
- SQL: SELECT, INSERT, UPDATE, DELETE
- JOINs und Subqueries
- Normalisierung (1NF, 2NF, 3NF)
- ER-Diagramme

**Prüfungsfokus:** SQL-Queries schreiben, Normalisierung verstehen

---

#### ⭐⭐⭐ LF4: Frontend (140h)
**SUPER WICHTIG - Praktische Prüfung!**
- HTML5 & CSS3
- JavaScript (ES6+, DOM, Events)
- React oder Vue.js Basics
- Responsive Design

**Prüfungsfokus:** UI bauen, Responsive, Framework-Grundlagen

---

#### ⭐⭐⭐ LF5: Backend (160h)
**SUPER WICHTIG - Die Logik!**
- REST APIs
- Node.js/Express oder Django
- Authentication & Authorization
- Datenbankintegration

**Prüfungsfokus:** APIs schreiben, Auth implementieren

---

#### ⭐⭐ LF6: Web-Sicherheit (100h)
**KRITISCH - Wird geprüft!**
- OWASP Top 10
- Input Validation & Sanitization
- SQL Injection, XSS, CSRF
- Verschlüsselung, Hashing

**Prüfungsfokus:** Sicherheitslücken erkennen, fixen

---

### BLOCK 2: PROFESSIONELLE SKILLS (40% der Prüfung)

#### 🔥 LF1: IT-Systeme (nur Netzwerk) (20h)
**Nur das Netzwerk-Teil!**
- TCP/IP, OSI-Modell
- IP-Adressierung, Subnetting
- DNS, DHCP
- Firewall-Grundlagen

**Prüfungsfokus:** Netzwerk-Troubleshooting

---

#### 🔥 LF10: Design Patterns (80h)
**WICHTIG für sauberen Code!**
- SOLID-Prinzipien
- Design Patterns (Singleton, Factory, Observer)
- Clean Code Prinzipien
- Refactoring

**Prüfungsfokus:** Code Quality, Pattern-Auswahl

---

#### 🔥 LF11: Testing (85h)
**WICHTIG - Test-getriebene Entwicklung!**
- Unit Tests (JUnit, pytest)
- Integration Tests
- TDD (Test-Driven Development)
- Mocking & Stubbing

**Prüfungsfokus:** Tests schreiben, QA-Prozesse

---

#### 🔥 LF12: Team & Git (85h)
**WICHTIG - Realität der Arbeit!**
- Git/GitHub (Grundlagen)
- Agile/Scrum
- Code Review Prozesse
- Versionskontrolle

**Prüfungsfokus:** Git-Workflow, Team-Zusammenarbeit

---

## 📊 PRÜFUNGSVERTEILUNG

```
Schriftliche Prüfung (60%):
  - LF2, LF3, LF4, LF5, LF6: ~80%
  - LF1 (Netzwerk), LF10, LF11, LF12: ~20%

Praktische Prüfung (40%):
  - Projekt mit Frontend + Backend
  - Database Design
  - Code Quality & Testing
  - Security Aspects
```
